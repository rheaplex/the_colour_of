About 'The Colour Of...'

The Colour Of... is Art

The Colour Of... is written using Ruby on Rails. It grabs images from web sites and extracts a colour palette from each one. This palette is then presented through the web interface.

The Colour Of... is Free Software

The code that runs The Colour Of... is free software licenced under the GNU Affero GPL version 3. There's a link to the source code on the front page. If anyone publishes a modified version of the software, they have to provide their own link.

The Colour Of... is Free Data

The data gathered by The Colour Of... is covered by the CC0 waiver of rights. There's a link to download the data on the front page. The data is updated only once a day, so there will be some lag behind what is seen on the site.
